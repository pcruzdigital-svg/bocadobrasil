Ao configurar aqui, o botão Publicar Site envia as alterações diretamente para o GitHub. O site atualiza em ~30 segundos para todos os visitantes.

Cria um token em GitHub → Settings → Developer settings → Personal access tokens → Fine-grained. Permissões necessárias: Contents → Read and write.
